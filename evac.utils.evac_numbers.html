Scripts to check the nature of numbers.

evac.utils.evac_numbers.check_type(n, typ, allow_array=True)[source]¶

Return true if input n is a given ‘type’. If n is an array, it checks if the dtype of the array matches the type. If the array is made up of many types, it raises an error.

Parameters:
  • n – Any object (N.ndarray, int, N.int32, N.float64...)
  • typ (str) – ‘int’ is any type or dtype resembling an integer ‘float’ is any type or dtype resembling a float
