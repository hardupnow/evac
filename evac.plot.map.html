evac.plot.map module¶

class evac.plot.map.Map[source]¶

Bases: evac.plot.figure.Figure

Plot a geographical map.

Similar to Birdseye, but without the ability to plot meteorological data. The main use is for plotting domains and other geographical maps.

make_domain_obj(input_objs)[source]¶

Turn WRFOut files or grids into domains.

plot_domains(domains, outdir, labels=None, latlons='auto', fname='domains.png', colour=0, res='h', proj='auto', pad=0.2)[source]¶

This method plots a figure of all domains specified.

Parameters:
  • domains (list,tuple) – list of WRFOut objects for each domain Largest domain should be index 0, if using auto settings
  • latlons (dict) – dictionary of Nlim,Elim,Slim,Wlim for plot
  • pad – multiply factor (0-1) for padding if latlons is ‘auto’
