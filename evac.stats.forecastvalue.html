evac.stats.forecastvalue module¶

class evac.stats.forecastvalue.ForecastValue(CLs=None, *args, **kwargs)[source]¶

Bases: evac.stats.detscores.DetScores

Compute FV, forecast value, as Buizza 2001.

Example that returns FV for exceeding 0.1 units (e.g., QPF) using the default range of cost/loss ratios:

FV = ForecastValue(fcst_arr=arr1,obs_arr=arr2,thresh=0.l,
overunder=’over’)
Then, access the FV for a given cost/loss ratio with:
FV.get(0.02)

This will return None is the cost/lost ratio doesn’t exist.
