evac.stats.detscores module¶

class evac.stats.detscores.DetScores(arr2x2=None, a=None, b=None, c=None, d=None, fcst_arr=None, obs_arr=None, thresh=None, overunder=None)[source]¶

Bases: object

Miscellaneous (and more simple) deterministic scores.

Using Buizza 2001, MWR, as follows:
  • a = observed and forecast
  • b = forecast but not observed
  • c = observed but not forecast
  • d = neither observed nor forecast

Users must supply the forecast and observation array (with threshold and over/under operation), or either arr or (a,b,c,d).

Parameters:
  • fcst_arr,obs_arr (N.ndarray) – Forecast and verification arrays, the same dimensions.
  • thresh (float) – threshold
  • overunder (str) – over, under, overeq, undereq, equal
  • arr2x2 (N.ndarray) – Array representing 2x2 matrix. Must be arranged so flatten array is equal to a tuple of (a,b,c,d).
  • a,b,c,d (int) – Number of elements in 2x2 matrix.
